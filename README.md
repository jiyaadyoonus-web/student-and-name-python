Student Performance Analysis System (Synthetic Data)

Project Overview The Student Performance Analysis System is a data analysis project that uses synthetic (artificial) student data to evaluate academic performance. It helps in identifying top-performing students, average students, and low-performing (fail) students based on defined criteria

Objectives

Analyze student marks and performance
Identify top performers and weak students
Generate insights using data analysis
Visualize performance trends using graphs
Dataset (Synthetic Data)

The dataset is artificially created and includes:

Student ID

Name

Subject marks (Math, Science, English, etc.)

Total marks

Average score

Grade Technologies Used

Python

NumPy

Pandas

Matplotlib / Seaborn Features

✔️ Generate synthetic student data

✔️ Calculate total and average marks

✔️ Assign grades automatically

✔️ Identify:

Top performers
Average students
Fail students
Data visualization (bar chart, line plot Example Analysis

Students with average ≥ 90 → Top performers

Students with average between 50–89 → Average

Students with average < 50 → Fail

How to Run Output

Student performance table

Summary statistics

Graphs showing performance distribution
